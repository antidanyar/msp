# Morpheme Structure Phonotactics: models & results

This repo contains the materials used for re-analysis of the data in Becker, Gouskova (2016, Linguistic Inquiry) by Kasenov (2026, Proceedings of the Society for Computation in Linguistics).

It's organized as follows:
* `data`: this folder contains the materials taken from Becker and Gouskova's study (their lexicon of Russian nouns, their experimental results, the predictions of their sublexical learner) and the feature decomposition matrices employed in the paper.
* `code`: this folder contains the main Jupyter Notebook pipeline
* `results`: this folder contains all the intermediate results files: experimental result file appended with my models' predictions, correlation values, AUC values, and so on.

The purpose of this repo is mostly to demonstrate the tools used for the results reported in the paper. For example, the notebook is not written with this particular folder structure in mind---if you want to run it, you will need to tinker file reading a bit locally.
